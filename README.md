**Properties**

- Commutative

```
a + b = b + a
```

- Associative

```
( a * b ) * c = a * ( b * a )
```

- Distributive Property

```
14 * 11 = 14 * ( 7 + 4 ) = ( 14 * 7 ) + ( 14 * 4 )
```
